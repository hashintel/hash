# What are custom blocks?

Custom blocks are user-developed blocks, not built by _HASH_, which can nevertheless be inserted into any [page](/guide/pages) in HASH.

# Building custom blocks

To develop your own block, you'll need to know how to code. More information about building your own blocks can be found in the [HASH Developer Docs](https://hash.dev/docs/blocks).
